Команда df

df используется в Linux для отображения информации о свободном месте на файловых системах. Она показывает общий объём, используемый объем и доступное свободное пространство дисков и разделов.

Примеры использования:

Отображение свободного места на всех файловых системах

df

Показать информацию в удобочитаемом формате (гига-, мега- и килобайты)

df -h

Подробная помощь (справка)

df --help

Справочная страница (man)

man df

Использование помощи и документации

Результат команды df --help

olgar@do4:~/DO4/src$ df --help
Usage:
 df [OPTION]... [FILE]...

Show information about the file system where each FILE resides, or all mounted systems if none given.

Mandatory arguments to long options are mandatory for short options too.
  -a, --all                 include pseudo, duplicate, inaccessible file systems
  -B, --block-size=SIZE     scale sizes by SIZE before printing them
  -H                        print sizes in powers of 1000 (for humans)
  -h, --human-readable      print sizes in human readable format (e.g., 1K 234M 2G)
  -k                        like --block-size=1K
  -m                        like --block-size=1M
  -P, --portability         use the POSIX output format
  -T, --print-type          print file system type
  -t TYPE                   limit listing to file systems of type TYPE
  -x TYPE                   exclude file systems of type TYPE
  -X DEVICE                 exclude file systems on device DEVICE
  -Z, --no-sync             do not invoke sync before getting usage info
      --total               produce a grand total
      --help                display this help text and exit
      --version             output version information and exit

Справочная страница (man df)

olgar@do4:~/DO4/src$ man df
DF(1)                           User Commands                          DF(1)

NAME
       df - report file system disk space usage

SYNOPSIS
       df [OPTION]... [FILE]

DESCRIPTION
       Report the amount of disk space available on the file system containing each file argument. If no file is given, then for each mounted file system.

       With no arguments, reports the amount of free and used space on each mounted file system.

OPTIONS
       -a, --all
              show all file systems including dummy ones

       -B, --block-size=SIZE
              specify the block size SIZE in bytes (default is 1K)

       -H, --si
              print sizes in powers of 1000 (for example, 'MB' means megabytes)

       -h, --human-readable
              print sizes in human readable format (e.g., 1K 234M 2G)

       -k     like --block-size=1K

       -m     like --block-size=1M

       -P, --portability
              use the POSIX output format

       -T, --print-type
              print file system type

       -t TYPE
              limit listing to file systems of type TYPE

       -x TYPE
              exclude file systems of type TYPE

       -X DEVICE
              exclude file systems on DEVICE

       -Z, --no-sync
              don't invoke sync before getting usage info

       --total
              produce a grand total

       --help
              display this help and exit

       --version
              output version information and exit

Таким образом, команда df позволяет удобно отслеживать состояние дискового пространства в вашей операционной системе.
